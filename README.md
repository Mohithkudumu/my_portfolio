# My Portfolio

A modern, fast, and responsive web portfolio built with React, Vite, and Tailwind CSS.

## Features
- Minimalist and clean design
- Fast loading and hot reloading powered by Vite
- Fully responsive styling with Tailwind CSS

## Tech Stack
- **Frontend Framework**: React + TypeScript
- **Styling**: Tailwind CSS
- **Bundler**: Vite

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Mohithkudumu/my_portfolio.git
   ```

2. Navigate into the project directory:
   ```bash
   cd my-mediocre-portfolio-main
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

## Live Demo
Check out the project running locally at `http://localhost:8080/`.
